# Course Name - README

This is the user manual how to set up a project for studying.


## File Structure
---
- **topics**
	- Here you write down the main topics of the syllabus index or the main sections of the course.
- **glossary**
	- This is for definitions you find while studying.
- **formulas**
	- This is for mathematical formulas, coding functionalities, and if/then flows.
- **processes**
	- This is to write down specific processes/workflows to follow, to reach a goal in practical life then.
- **tools**
	- This is the list of tools that are available to work with the learned in the practical live (image resizing, CRM, testing tool, etc.).
- **methodologies**
	- This is is a comparison and a decision helper for use the learned in the practical live, like a protocol to take the right decisions at a specific point/topic.
- **resources**
	- Here you link online resources or notes to have them documented and navigate fast.
- **overview**
	- Here you document the data about the course and the exam.


## Tools
---
This is the list with tools you need for working efficient:
- Obsidian.
- Ahmni app (iPadOS) (full version).
- Anki Flashcards app (full version).
- Perplexity AI (for research the web - better version of Google).
- NotebookLM (for creating podcasts).
- ChatGPT (for creating content, tests, prompts, etc.).
- Claude (for IT content or more specific language abilities).
- Notebook and pen in your pocket for writing down stuff while not studying.


## Todos
---
- [ ] Copy this studies directory into /1-projects directory in your Obsidian vault.
- [ ] Rename this directory by the name of the course.
- [ ] Fill out the data in overview file.
- [ ] Create the project in your AI tools.
- [ ] Search for course materials (PDFs, study guides, videos, etc.) and link them in resources file.
- [ ] If you do the course on a video platform like Udemy, capture screen shots of the lessons list.
- [ ] Copy the study-template directory on your disk and put in all the downloaded materials and the screenshots in there.
- [ ] Add the syllabus or the images into ChatGPT and create the course plan by the prompt:
```
Create a course structure out of the added material, that I can use for me structure to study.
```
- [ ] Add the topics to the topics file.
- [ ] Pass all the course sources into NotebookLM (PDFs, videos, etc.)
- [ ] Add the syllabus and/or the screenshots of the course lessons into ChatGPT.
- [ ] Create the course structure and the prompts for NotebookLM by the prompt:
```
Create a learning schedule by the course plan. I've added all the materials to Google NotebookLM to create podcasts about it. Create the prompts I can pass to NotebookLM for creating the podcasts based on your schedule.
```
- [ ] In NotebookLM select all sources and create notes out of the ChatGPT prompts and save them as source.
- [ ] Refine the topics file by the result of ChatGPT.
- [ ] Unselect the sources and select only the note and the main material, out of which you would like to create the podcast by the prompt:
```
Focus only on the check marked sources. You be a very professional on this topic and you are very amazing in teaching people, that are studying this topic to pass the exam and get the certification. You are on point and don't use much fill words or small talk. Your stuff is easy to understand and the pupils can easily follow you.
```
- [ ] Download and save them on disk, add the tracks into your music app.
- [ ] Open Ahmni app and copy the StudyTemplate canvas and rename it with the name.
- [ ] Add the course topics inside the canvas. Explanation is in Ahmni setup file.
- [ ] Open AnkiWeb and search for predefined decks for the course (if there are none, you have to create cards while the course). Do not sort the cards! It is important that they're come like they are.
- [ ] Check if the settings of the deck are set up like described in Anki setup file.
- [ ] Create an entry test for the very first step of your studies by pasting the topics and the material into Claude AI:
```
This is my course material.
Assume I've finished the course and create 20 questions for me to answer on the whole course topics.
Start from general and get more specific.
Create a PDF file with the questions and answers.
```
- [ ] For every main level topic in the topics file, create a test for every time a new topics begins with Claude AI (add your course material as context) by the prompt:
```
This is my course material.
Assume I've finished the topic and create 20 questions for me to answer on the course topic "TOPIC NAME".
Start from general and get more specific.
Create a PDF file with the questions and answers.
```

If all boxes are checked, you can start to study. The studying workflow is described in [[how-to-study]].
